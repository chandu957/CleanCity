#  CleanCity – Smart Garbage Reporting System

## Problem
Many cities face garbage management issues because people lack an easy way to report them.

##  Solution
CleanCity allows users to report garbage issues by entering location and description.

##  Features
- Submit garbage reports
- View all reports
- Simple and user-friendly UI

##  Tech Used
- HTML
- CSS3
- JavaScript

##  Impact
Helps keep cities clean and encourages public participation.
